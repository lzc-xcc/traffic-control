1

交叉口渠化设计与相位方案

本设计将原五路畸形交叉口改造为双节点联动协调系统：

  • 节点1（主十字路口）：DAK1+100，标准十字交叉，5车道设计
  • 节点2（T型交叉口）：B路接入点，与主路形成T型交叉
  • 间距：两节点间距60米，实现绿波协调控制

相位方案设计：采用四相位控制方案，包含南北直行、东西直行、南北左转、东西左转四个相位。

2

各车道组直行当量计算

根据交通调查数据，各转向流量如下表所示：

直行当量系数：左转车辆当量系数E_L=1.5，右转车辆当量系数E_R=1.0

V_e = V_s + 1.5 × V_l + V_r
3

流量比分析与关键车流确定

计算各车道组的流量比（Y值）：

车道组流量比Y_i
南北直行0.18
东西直行0.22
南北左转0.12
东西左转0.084

总流量比 Y = ΣY_i = 0.604

Y = 0.604 < 0.9，满足信号控制设计要求

4

确定黄灯时间和全红时间

A = t + v/(2g)
其中：t=1.0s，v=50km/h，g=2.5m/s²

计算结果

黄灯时间 A = 3s

全红时间 r = 2s

绿灯间隔时间 I = A + r = 5s

5

确定信号损失时间

每相位损失时间 L_i = I - A = 5 - 3 = 2s

启动损失时间 = 3s

L = n × L_i + 启动损失 = 4 × 2 + 3 = 11s

总损失时间 L = 11s

6

确定信号周期时长

C_0 = L / (1 - Y) = 11 / (1 - 0.604) ≈ 27.8s

实际周期：取标准周期 C = 60s（考虑行人过街需求和绿波协调）

最终信号周期 C = 60s

7

绿灯时间分配

G_i = (C - L) × (Y_i / Y)

8

行人过街时间检验

t_p = S / v_p + t_r
其中：S=20m，v_p=1.2m/s，t_r=3s

行人最小绿灯时间 t_p = 20/1.2 + 3 ≈ 19.7s

相位2东西直行绿灯时间=24s ≥ 19.7s ✓ 满足要求

相位1南北直行绿灯时间=12s < 19.7s → 需优化相位方案
